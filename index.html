The Arduino FT 817 Interface

The Arduino FT 817 Interface contains two components:

  1. A library for accessing the FT 817 from an Arduino
  2. A remote control program using this library

Features of the Remote Controller
  • Maidenhead location display
  • Distance to current repeater
  • Featuring bandplan information (using <XML>Bandplan).
  • Multiple channel watch
  • Switching channels
  • Automatic mode selection based on bandplan
  • Frequency browsing using a rotary encoder
