Arduino FT 817
--------------
This is a standalone library for controlling the YAESU FT 817 
using the famous arduino. It also features some examples on what to do.

Released under GPLv3.

Installation Instructions
-------------------------
* Copy this whole current directory to
  ~/Documents/Arduino/libraries/FT817

References
----------
* Documentation of the serial stuff: http://www.ka7oei.com/ft817_meow.html
* 2nd version of the lib: http://www.alain.it/2012/02/10/ft-817-automatic-loop/
* 1st version of the lib: http://code.google.com/p/qrptracker/
